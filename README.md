# OmniChem-Universal-Chemical-Property-Extraction-Analysis-System
README: OmniChem: Universal Chemical
Property Extraction & Analysis System
Welcome to OmniChem. This project enables automated chemical property
retrieval using Python. No SMILES knowledge required.
🛠 Prerequisites
Install the necessary chemical informatics libraries via pip:
pip install rdkit-pypi pubchempy pandas cclib
📂 File Structure
main.py: The core analysis engine.
outputs/: Directory where PDF and CSV reports are saved.
logs/: Store your .out files from ChemCompute here.
🚀 Getting Started
1. Run the script python main.py.
2. Enter any chemical name (e.g., "Aspirin" or "Caffeine").
3. Review the generated molecular summary and atom breakdown.
✨ Core Features
Universal Lookup: Direct connection to PubChem (110M+ compounds).
3D Preparation: Automatic addition of Implicit Hydrogens for accurate
modeling.
Hybridization Mapping: Detailed atomic-level analysis for reactivity prediction.
